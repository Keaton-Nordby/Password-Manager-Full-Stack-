Password Manager Full-Stack App

A full-stack password manager built with React, Node.js/Express, and MySQL. This app allows users to securely add and store passwords with titles, encrypts passwords before storing them in the database, and provides easy and secure access.

Features
- Add a password with a title (e.g., Facebook, Gmail)
- Passwords are encrypted before being stored in the database
- Secure and easy access to stored passwords
- Simple and intuitive frontend with React
- Backend powered by Node.js and Express
- MySQL database stores encrypted passwords

Tech Stack
- Frontend: React.js, Axios
- Backend: Node.js, Express.js

Database: MySQL
Encryption: bcrypt (or preferred encryption library)
